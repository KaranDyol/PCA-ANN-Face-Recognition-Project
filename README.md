# PCA-ANN Face Recognition System
This project implements a Face Recognition system using **Principal Component Analysis (PCA)** for feature extraction and an **Artificial Neural Network (ANN)** for classification.

## Project Overview
- **Dataset:** Synthetic face image data stored in CSV format.
- **Feature Extraction:** PCA (Eigenfaces) reduced 4096 pixels to 100 components.
- **Classifier:** Multi-Layer Perceptron (ANN) with 1024 hidden neurons.
- **Results:** Achieved high recognition accuracy on the test set.

## Files
- `pca_ann_project.py`: Main implementation script.
- `train_faces.csv` & `test_faces.csv`: Dataset files.
- `pca_model.pkl` & `ann_model.pkl`: Trained models.
